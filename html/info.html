MUSIC-TO-COLOR
A LIVE CHROMESTHESIA VISUALIZER



MAIN CONCEPT:
MUSIC-TO-COLOR allows you to play a musical instrument via MIDI and get a color-based representation of each note in real time.
HELP:
MAIN MENU - to access the MENU, move your cursor to the left edge of the screen.
INSTRUMENT - the INSTRUMENT tab lets you see connected MIDI devices and lets you test for note input
COLOR - the COLOR tab allows you to change the display color of each note
STYLE - the STYLE tab is for customizing different aspects of the display that doesn't relate to color
INFO - a link back to this page
