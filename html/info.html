MUSIC-TO-COLOR
A LIVE CHROMESTHESIA VISUALIZER



MAIN IDEA:
MUSIC-TO-COLOR allows you to play a musical instrument via MIDI and get a color-based representation of each note in real time.
HELP:
MAIN MENU - to access the MENU, move your cursor to the left edge of the screen.
INSTRUMENT - see connected MIDI devices
COLOR - change the display color of each note
STYLE - customize different aspects of the display such as size and location of each color
INFO - a link back to this page
