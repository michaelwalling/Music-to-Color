MUSIC-TO-COLOR
A LIVE CHROMESTHESIA VISUALIZER



MAIN IDEA:
MUSIC-TO-COLOR allows you to play a musical instrument via MIDI and get a color-based representation of each note in real time.
HELP:
MENU - to access, move your cursor to the left edge of the screen
DEVICES - see connected MIDI devices and their information
COLORS - randomize, transpose, choose a color template, or customize each individual note color
SETTINGS - customize different aspects of the display such as size and location of each color
INFO - a link back to this page
